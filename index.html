How does the World Wide Web work?
When you sit infront of your computer you get on the browser e.g. Chrome,Internet Explorer etc .
The browser sends a signal requesting the files via the internet to the servers.
The servers send back the requested files to the computer.

This is a basic outline on how the World Wide Web work.

WHITESPACE
This is the space between lines.
This is created by < br > line break, this is a void, there is no content no closing tag.
Then there is < p> content < /p>. < paragraph> when used it creates a space between lines.
